# Mini Resume Management API 
## Features
- Upload Resume (PDF/DOC/DOCX)
- Store Candidate Metadata
- Filter by Skill, Experience, Graduation Year
- Get Candidate by ID
- Delete Candidate
## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite
## Setup Instructions
1. Create Virtual Environment
2. Install Dependencies
3. Run Server
4. Open Swagger
